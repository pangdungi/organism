# 라이프 플래너 — 단계별 진행

## 1단계 (지금 완료)
- [x] 프로젝트 폴더 구성
- [x] Vite로 실행 가능한 뼈대 (index.html, src/main.css, src/main.js)
- [x] 헤더 + 간단한 메인 문구만 표시

**실행 방법**
```bash
cd "라이프플래너 1"
npm install
npm run dev
```
브라우저에서 `http://localhost:5173` 열기.

(npm install 시 캐시 권한 오류가 나면: `npm cache clean --force` 후 다시 시도)
